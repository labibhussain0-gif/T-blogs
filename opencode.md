# Open Code Instructions for T-blogs

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Rendering**: Static Site Generation (SSG) MUST be used. Ensure `output: 'export'` is in `next.config.mjs` or `next.config.ts`.
- **Data Fetching**: All data must be fetched at build time. No dynamic server functions (`cookies()`, `headers()`, `searchParams`).
- **Niche**: Technical and AI blog.
- **Design Standard**: Clean, minimal UI inspired by high-end tech blogs. Orange/Red accent colors, robust grid layouts for articles.
